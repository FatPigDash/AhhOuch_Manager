# AhhOuch_Manager 多人使用需求



## 幹部系統

1. 軟體名稱：AhhOuch_Manager
2. 使用者：多人使用，針對店家幹部
3. 用於手機，針對IOS系統、同時兼容安卓系統
   1. 使用者只會用手機或是平板，不會用到電腦。
   1. 每個使用者的資料是獨立的。
   1. 圖片預設使用2000張。
4. 開發資訊：
   1. 開發者不懂程式。
   2. 使用電腦開發：
      1. 電腦：WIN11，系統語系為繁體中文、日文、英文。
      2. VS code

### 執行目標：給幹部方便整理美容師資訊及發佈班表。

1. 開發者不自己使用一台電腦24小時開著當伺服器。
2. 使用者如果有手機+平板同步使用的需求，自行利用帳號備份功能，裝置A上傳、裝置B下載，利用手動進行資料的傳輸。
