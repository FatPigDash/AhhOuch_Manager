# AhhOuch 多人使用需求

### 目標使用者

1. 軟體名稱：AhhOuch
2. 使用者：多人使用，針對消費者
3. 用於手機，分為IOS系統和安卓系統。
   1. 僅開發者在開發過程需要讓手機和電腦的資料串聯。
   2. 每個使用者的資料是獨立的。

4. 開發資訊：
   1. 開發者不懂程式。
   2. 電腦：WIN11，系統語系為繁體中文、日文、英文。
   3. VS code

### 執行目標：為達到營利目的，直接把客人系統拆出來，針對免費版本給予受限制的使用功能，付費版本開放完整功能。。

#### 免費版本的功能：

1. 養身館卡片僅能建立1個。
2. 美容師卡片
   1. 美容師簡介僅能上傳一張圖片。
   2. 心得功能完整。
