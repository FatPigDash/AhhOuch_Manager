# 手機版前端（架構保留）

依需求 G1，介面需支援電腦與手機。**M7 已確認採「響應式單一程式」做法**：

- 手機版**不另建獨立專案**，而是由 `frontend/desktop/` 同一套 Vue 程式以
  RWD（響應式設計，`@media (max-width: 640px)`）自適應手機螢幕。手機瀏覽器
  開同一網址即自動切換為手機版面。
- 好處：單一程式碼、單點維護（非程式背景的維護者不需同步改兩套）。

此資料夾**保留作未來擴充用**：若日後手機需要與電腦截然不同的操作流程（而非
僅版面調整），可在此另建獨立 Vite + Vue 專案，並於 `backend/config.py`、
`backend/main.py` 增加對應的靜態檔掛載路徑與裝置判斷。
